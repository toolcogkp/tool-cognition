Detailed Description

State machine Event structure: Manipulation This event is invoked by Olivia state machine when in the Active state. Perform sequences of action to manipulate object to a given target goal.

State machine will handle reaction based on target position and perceived object position. The objective is to bring the object to the target.

when the object is out of reach for Olivia's arm, perform search for viable tool in surroundings, use the monte carlo search tree algorithm to determine how to use the percieved tool as an extension of its arm, and finally executes the collision-free planning.
